# Airfoil Coordinate Data

This directory contains coordinate files used by the XFOIL-kernel regression
baseline cases.

Files:

- `sc1095_selig.dat`: Sikorsky SC1095 airfoil coordinates in Selig-style
  format. Use this as the first rotorcraft-relevant coordinate-file baseline.
- `stress/`: coordinate files used to exercise thicker, highly cambered, and
  higher-curvature geometries during modernization safety checks.

Keep source/provenance notes with each added file when available.
